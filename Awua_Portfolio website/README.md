# Engr. Dr. J. T. Awua - Professional Portfolio

This is the professional portfolio website for Engr. Dr. J. T. Awua, a Senior Lecturer, Thermofluid & Energy Engineer, and Plant Valuation Expert based in Nigeria.

## 🚀 Live Demo
[https://awuajt-portfolio.web.app/](https://awuajt-portfolio.web.app/)

## 🛠️ Technologies Used
- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom design system, responsive layout (Flexbox/Grid), and modern animations (glassmorphism, gradients).
- **JavaScript**: Smooth scrolling, mobile navigation toggle, and Intersection Observer for scroll animations (fade-ins, skill bar progress).
- **Firebase Hosting**: High-speed global deployment and CDN.

## 🌟 Key Features
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Modern Aesthetics**: A professional, engineering-focused design utilizing a deep navy blue and vibrant orange color palette.
- **Interactive Elements**: Micro-animations on hover, smooth section transitions, and dynamic skill progress bars.
- **Custom 404 Page**: A unique, branded "Page Not Found" experience with CSS animations.
- **SEO Optimized**: Open Graph and Twitter Card specific meta tags for elegant social sharing.

## 📂 Project Structure
- `index.html`: The main entry point showcasing the profile, expertise, experience, and portfolio.
- `style.css`: The external stylesheet containing the design system, utility boundaries, styling, and animations.
- `script.js`: External JavaScript managing user interactions, navbar styling on scroll, and viewport intersection events.
- `404.html`: Custom error page for missing routes.
- `firebase.json` & `.firebaserc`: Configuration rules mapping for Firebase Hosting.
